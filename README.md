#  Banking System Backend

A robust RESTful API developed using **Spring Boot** to handle core banking functionalities. This project demonstrates clean code practices and logical implementation of financial transactions.

##  Features
*   **Account Management:** API endpoints to create and fetch account details.
*   **Transaction Logic:** Robust implementation of `credit` and `debit` operations with real-time balance updates.
*   **Data Persistence:** Uses Spring Data JPA with MySQL for reliable data storage.
*   **Layered Architecture:** Follows the Controller-Service-Repository pattern for scalability.

##  Tech Stack
*   **Language:** Java 17+
*   **Framework:** Spring Boot 3.x
*   **Database:** MySQL / Hibernate
*   **Build Tool:** Maven
